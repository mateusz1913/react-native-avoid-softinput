Use `setAdjustNothing`, to set `android:windowSoftInputMode` attribute to `adjustNothing` value.

### Example

```ts
import { AvoidSoftInput } from "react-native-avoid-softinput";

AvoidSoftInput.setAdjustNothing();
```
