Use `setAdjustUnspecified`, to set `android:windowSoftInputMode` attribute to `adjustUnspecified` value.

### Example

```ts
import { AvoidSoftInput } from "react-native-avoid-softinput";

AvoidSoftInput.setAdjustUnspecified();
```
